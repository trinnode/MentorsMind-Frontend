# Learner Components

Session history and learning analytics components for learners.

## Components

### HistoryList
Displays complete session history with status, ratings, and outcomes.

### LearningAnalytics
Comprehensive analytics dashboard showing:
- Skill development progress
- Mentor interaction statistics
- Session frequency charts
- Spending analytics
- Learning velocity metrics

### TimeInvested
Visual card showing total time invested, average session duration, and total sessions.

## Usage

```tsx
import SessionHistory from '../pages/SessionHistory';

// In your router
<Route path="/history" element={<SessionHistory />} />
```

## Features

- Complete session history list
- Learning analytics dashboard
- Total time invested tracking
- Skill development metrics
- Mentor interaction statistics
- Session frequency charts
- Spending analytics
- Session outcome tracking
- Learning velocity metrics
- CSV report generation
